# Aliases

### Set an Alias

`Usage: ioctl alias set ALIAS ADDRESS`

```
➜  ioctl alias set test io1l3wc0smczyay8xq747e2hw63mzg3ctp6uf8wsg
set
```

### Remove an Alias

`Usage: ioctl alias remove ALIAS`

```
➜  ioctl alias remove frank
frank is removed
```
